# Yoobic Ionic Task

To run the app:
<ol>
  <li>Clone or Download the repo</li>
  <li>Run ```npm i```</li>
  <li>Navigate to http://localhost:8100/login in Chrome</li>
  <li>Open the dev tools console and turn on Phone emulator view</li>
  <li>You can use any credentials to login.</li>
  <li>Enjoy!</li>
</ol>
